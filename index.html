GalliumOS Kernel Installer (See code)

I love Linux. And I love distro-hopping (going from one operating system to another). One of the problems I found with distro-hopping was having to manually download and install the required files so my computer can work (audio, speed, etc.). So I built a script that does all of that for you (in the language BASH, which I learned as I was working - roughly 20 minutes). I finished the script in one-sitting and shared it.

Olluro: School Assignments (See DEMO)

Sometimes you're absent from class and you miss the classwork and the assignments. Olluro would be a place where not only teachers but classmates can upload classwork and homework assignments so you can always stay updated on what's going on.

Flat UI for Materialize (See code)

Materialize is an amazing CSS framework with tons of cool components that the more-popular Boostrap doesn't have (tons of different card styles and configurations, preloaders, etc). But what if you wanted to use Materialize without needing to restrict yourself to the Material touch? That's why I built...simpleMaterializeCSS. It relies on the Materialize code but changes a few things to make it more Flat-UI-like.
